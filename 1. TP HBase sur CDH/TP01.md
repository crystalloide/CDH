# TP01 : Introduction à HBase

## Présentation

Nous allons explorer l'environnement technique (l'instance HBase est incluse dans l'image Cloudera Quickstart que nous allons lancer via VirtualBox)

---

## Exploration de l'environnement


a. Si besoin( normalement non), exécutez la commande suivante dans le terminal pour démarrer Cloudera Manager :

```bash
$ sudo /home/cloudera/cloudera-manager --express
```

b. Ouvrez un navigateur : 

```
localhost:7180
```

et saisissez les identifiants suivants, puis cliquez sur **Log In** :

```
Username: cloudera
Password: cloudera
```

c. Cliquez sur le menu déroulant **Cloudera Quickstart** et sélectionnez **Start** pour démarrer Cloudera.

d. Les services démarrent — cliquez sur **Close** une fois terminé.

**1. Lancer l'interface Hue.**

a. Ouvrez un nouvel onglet dans le navigateur et saisissez :

```
localhost:8888
Username : cloudera
Password : cloudera
```

b. Vérifiez que tous les exemples de l'étape sont installés, puis cliquez sur **Next**.

**2. Dans le terminal Cloudera Quickstart, exécutez les commandes suivantes pour se connecter à l'instance HBase :**

```bash
export HADOOP_USER_NAME=hdfs
hbase shell
```

**3. Ouvrez un nouvel onglet dans le navigateur et saisissez `localhost:60010` pour afficher la page d'accueil du serveur Master dans l'interface HBase UI.**

**4. Ouvrez un nouvel onglet dans le navigateur et saisissez `localhost:60030` pour afficher la page d'accueil du RegionServer dans l'interface HBase UI.**

**5. Laissez Cloudera Manager, l'instance HBase et l'interface HBase UI ouverts pour la pratique suivante.**
