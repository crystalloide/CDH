# TP03 : les APIS HBase

## Présentation

Nous allons configurer le serveur REST HBase, puis invoquer les API d'administration et API Client via l'outil Postman.

---

## Configuration de l'environnement

### Présentation

Dans cette pratique, nous allons ajouter un rôle au serveur REST HBase et configurer l'URL proxy pour invoquer les APIs via Postman.

**Durée :** 30 minutes

**Prérequis :** Vous devez avoir complété le TP02

---

### Tâches

**I. Configurer le serveur REST Proxy dans HBase.**

1°) Sur la page d'accueil de Cloudera Manager, cliquez sur **Clusters** et sélectionnez **HBase**.

2°) Dans le tableau de bord HBase de Cloudera Manager, cliquez sur **Actions** et sélectionnez **Add Role Instances**.

3°) Pour ajouter le rôle du serveur REST Proxy à l'instance HBase, cliquez sur **Select Hosts** tout à droite sur **HBase REST Server**.

4°) Dans la fenêtre d'ajout d'hôte, sélectionnez le nom d'hôte dans la liste **quickstart.cloudera**, cochez la case et cliquez sur **OK**.

5°) Vérifiez que l'hôte a bien été ajouté au serveur REST HBase, puis cliquez sur **Continue**.

6°) Vérifiez que le rôle HBase Rest Server a bien été assigné dans le tableau de bord HBase de Cloudera Manager.

7°) Dans le tableau de bord HBase, cliquez sur **Actions** et sélectionnez **Start**.

8°) Cliquez sur **Start** pour démarrer le serveur REST HBase.

9°) Attendez que l'instance soit démarrée.

10°) Vérifiez que l'instance du serveur REST HBase a démarré, puis cliquez sur **Close**.

11°) Dans le tableau de bord HBase de Cloudera Manager, vérifiez que le serveur REST HBase est bien démarré.

12°) Dans le terminal Cloudera Quickstart, exécutez la commande suivante pour démarrer le serveur REST Proxy :

```bash
hbase rest start --p 8080 --infoport 8090
```

13°) Vérifiez que le statut du serveur REST Proxy s'affiche comme démarré.

14°) Téléchargez Postman via l'URL suivante et installez-le sur votre système local :
https://dl.pstmn.io/download/latest/win64

text

15°) Ouvrez l'outil Postman sur le système local et cliquez sur l'icône **+**.

16°) Une console s'ouvre dans Postman pour envoyer des requêtes et invoquer l'API REST Proxy HBase.

**II. Laissez le serveur REST HBase et l'outil Postman ouverts pour la pratique suivante.**
