# TP03 : HBase APIs

## Présentation

Nous allons configurer le serveur REST HBase, puis invoquer les APIs d'administration et les APIS Client via l'outil Postman.

---

## Configuration de l'environnement

### Présentation

Dans cette pratique, nous allons ajouter un rôle au serveur REST HBase et configurer l'URL proxy pour invoquer les APIs via Postman.

**Durée :** 30 minutes

**Prérequis :** Vous devez avoir complété le TP02

---

### Tâches

**1. Configurer le serveur REST Proxy dans HBase.**

a. Sur la page d'accueil de Cloudera Manager, cliquez sur **Clusters** et sélectionnez **HBase**.

b. Dans le tableau de bord HBase de Cloudera Manager, cliquez sur **Actions** et sélectionnez **Add Role Instances**.

c. Pour ajouter le rôle du serveur REST Proxy à l'instance HBase, cliquez sur **Select Hosts** tout à droite sur **HBase REST Server**.

d. Dans la fenêtre d'ajout d'hôte, sélectionnez le nom d'hôte dans la liste **quickstart.cloudera**, cochez la case et cliquez sur **OK**.

e. Vérifiez que l'hôte a bien été ajouté au serveur REST HBase, puis cliquez sur **Continue**.

f. Vérifiez que le rôle HBase Rest Server a bien été assigné dans le tableau de bord HBase de Cloudera Manager.

g. Dans le tableau de bord HBase, cliquez sur **Actions** et sélectionnez **Start**.

h. Cliquez sur **Start** pour démarrer le serveur REST HBase.

i. Attendez que l'instance soit démarrée.

j. Vérifiez que l'instance du serveur REST HBase a démarré, puis cliquez sur **Close**.

k. Dans le tableau de bord HBase de Cloudera Manager, vérifiez que le serveur REST HBase est bien démarré.

l. Dans le terminal Cloudera Quickstart, exécutez la commande suivante pour démarrer le serveur REST Proxy :

```bash
hbase rest start --p 8080 --infoport 8090
```

m. Vérifiez que le statut du serveur REST Proxy s'affiche comme démarré.

n. Téléchargez Postman via l'URL suivante et installez-le sur votre système local :
https://dl.pstmn.io/download/latest/win64

text

o. Ouvrez l'outil Postman sur le système local et cliquez sur l'icône **+**.

p. Une console s'ouvre dans Postman pour envoyer des requêtes et invoquer l'API REST Proxy HBase.

**2. Laissez le serveur REST HBase et l'outil Postman ouverts pour la pratique suivante.**
