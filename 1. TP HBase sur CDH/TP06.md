## Pratique 4-2 : Export des données de HBase vers PIG et de PIG vers HBase

### Présentation

Dans cette pratique, vous apprendrez à exporter les données depuis HBase vers PIG et depuis PIG vers HBase.

**Durée :** 30 minutes

**Prérequis :** Vous devez avoir complété la pratique de la Leçon 4-1.

**Scénario :** Export des données depuis HBase vers PIG et depuis PIG vers HBase.

---

### Tâches

**1. Créer une table dans le shell HBase.**

a. Dans le shell HBase, exécutez la commande suivante pour créer une table :

```hbase
create 'custord', 'custord_data'
```

b. Dans le shell HBase, exécutez les commandes suivantes pour ajouter des données dans la table :

```hbase
put 'custord', 1, 'custord_data:id', '1'
put 'custord', 1, 'custord_data:name', 'Richard'
put 'custord', 1, 'custord_data:zipcode', '12'
put 'custord', 2, 'custord_data:id', '2'
put 'custord', 2, 'custord_data:name', 'Melissa'
put 'custord', 2, 'custord_data:zipcode', '18'
put 'custord', 3, 'custord_data:id', '3'
put 'custord', 3, 'custord_data:name', 'Ann'
put 'custord', 3, 'custord_data:zipcode', '61'
```

c. Dans le terminal Cloudera Quickstart, exécutez la commande `pig` pour vous connecter au shell Grunt :

```bash
pig
```

d. Dans le shell Grunt, exécutez la commande suivante pour charger les données depuis HBase :

```pig
cust_data = LOAD 'hbase://custord' USING org.apache.pig.backend.hadoop.hbase.HBaseStorage('custord_data:id, custord_data:name, custord_data:zipcode') AS (id:int, name:chararray, zip:chararray);
```

e. Dans le shell Grunt, exécutez la commande suivante pour afficher le contenu chargé :

```pig
dump cust_data
```

f. En cas de succès, vérifiez que le contenu des tables s'affiche dans le shell Grunt.

g. Dans le shell Grunt, exécutez la commande suivante pour filtrer les données selon le critère donné :

```pig
filter_custdata = FILTER cust_data BY zip == '18';
```

h. Dans le shell Grunt, exécutez la commande suivante pour afficher les données filtrées :

```pig
dump filter_custdata
```

i. En cas de succès, les données filtrées s'affichent dans le shell Grunt.

j. Dans le shell Grunt, exécutez la commande suivante pour stocker les données filtrées dans HBase :

```pig
STORE filter_custdata INTO 'hbase://custord' USING
org.apache.pig.backend.hadoop.hbase.HBaseStorage('custord_data:id, custord_data:name, custord_data:zipcode');
```

k. Dans le shell HBase, exécutez la commande suivante pour afficher le contenu de la table :

```hbase
scan 'custord'
```

l. En cas de succès, vérifiez que le contenu de la table avec les données filtrées a bien été ajouté à la table d'origine.

**2. Laissez le shell HBase et le terminal Cloudera Quickstart ouverts pour la pratique suivante.**
