## Importation des données depuis HDFS vers HBase

### Présentation

Dans cette pratique, vous apprendrez à importer une table depuis HDFS vers HBase.

**Durée :** 30 minutes

**Prérequis :** Vous devez avoir complété la pratique de la Leçon 4-2.

---

### Tâches

**1. Importer le fichier `emp.csv` dans HDFS.**

a. Téléchargez l'exemple de fichier `emp.csv` fourni ci-dessous et sauvegardez-le sur votre machine locale.
emp.csv

text

b. Ouvrez l'invite de commande et, depuis le répertoire de téléchargement, chargez le fichier `emp.csv` dans HDFS :

```bash
docker cp <chemin>/emp.csv <nom_du_conteneur_docker>:/
```

c. Dans le terminal Cloudera Quickstart, exécutez la commande suivante pour copier le fichier `emp.csv` dans HDFS :

```bash
hadoop fs -put /emp.csv /user/root
```

d. Dans le shell HBase, exécutez la commande suivante pour créer une table :

```hbase
create 'emp_data1','cf1'
```

e. Dans le terminal Cloudera Quickstart, exécutez la commande suivante pour copier la table depuis HDFS vers HBase :

```bash
hbase org.apache.hadoop.hbase.mapreduce.ImportTsv \
  -Dimporttsv.separator=',' \
  -Dimporttsv.columns='HBASE_ROW_KEY,cf1:ename,cf1:designation,cf1:manager,cf1:hire_date,cf1:sal,cf1:deptno' \
  emp_data1 /user/root/emp.csv
```

f. En cas de succès, vérifiez que toutes les données ont bien été écrites dans HBase.

g. Dans le shell HBase, exécutez la commande suivante pour afficher le contenu de la table :

```hbase
scan 'emp_data1'
```

h. En cas de succès, vérifiez que le contenu de la table s'affiche dans HBase.

**2. Déconnectez-vous du shell HBase avec la commande `quit` :**

```hbase
quit
```

**3. Fermez les terminaux Hive, Pig et Cloudera Quickstart.**
