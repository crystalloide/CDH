## Importation des données depuis HDFS vers HBase

### Présentation

**Durée :** 30 minutes

**Prérequis :** Vous devez avoir complété le TP06

---

### Tâches

**I. Importer le fichier `employes.csv` dans HDFS.**

1°) Téléchargez l'exemple de fichier `employes.csv` fourni dans /donnees et sauvegardez-le sur votre machine locale.
employes.csv

text

1°) Ouvrez l'invite de commande et, depuis le répertoire de téléchargement, chargez le fichier `employes.csv` dans HDFS :


2°) Dans le terminal Cloudera Quickstart, exécutez la commande suivante pour copier le fichier `employes.csv` dans HDFS :

```bash
hdfs dfs -put /employes.csv /user/root
```

3°) Dans le shell HBase, exécutez la commande suivante pour créer une table :

```hbase
create 'employes_donnees','cf1'
```

4°) Dans le terminal Cloudera Quickstart, exécutez la commande suivante pour copier la table depuis HDFS vers HBase :

```bash
hbase org.apache.hadoop.hbase.mapreduce.ImportTsv \
  -Dimporttsv.separator=',' \
  -Dimporttsv.columns='HBASE_ROW_KEY,cf1:ename,cf1:designation,cf1:manager,cf1:date_embauche,cf1:sal,cf1:departementID' \
  employes_donnees /user/root/employes.csv
```

5°) En cas de succès, vérifiez que toutes les données ont bien été écrites dans HBase.

6°) Dans le shell HBase, exécutez la commande suivante pour afficher le contenu de la table :

```hbase
scan 'employes_donnees'
```

7°) En cas de succès, vérifiez que le contenu de la table s'affiche dans HBase.

**II. Déconnectez-vous du shell HBase avec la commande `quit` :**

```hbase
quit
```

**III. Fermez les terminaux Hive, Pig et Cloudera Quickstart.**
